# Netflix Movies & TV Shows Analysis  
# Overview 
Exploratory analysis of Netflix content trends (2010-2021).  

## Tools Used  
- Python (Pandas, Seaborn, Plotly)  
- Jupyter Notebook  

## Key Insights  
- Movies dominate Netflix (75%), but TV shows are rising.  
- The US produces 40% of all content.
